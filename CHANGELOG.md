# Changelog

All notable changes to the **Wolfclaw** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0-Sovereign] - 2026-02-26

The "Sovereign" release marks the transition from private experimentation to a professional, agentic platform.

### Added
- **Sovereign UI**: A high-premium dashboard for visual orchestration.
- **Multiplexing Engine**: Support for multiple concurrent agents and providers (NVIDIA NIM, Ollama).
- **Session Persistence**: Token-based authentication and workspace isolation.
- **RAG Engine**: Local knowledge base integration for document-aware agents.
- **War Room Orchestrator**: Multi-agent coordination logic.
- **International Standard Docs**: README, Guide, Security, and Contributor guidelines.

### Changed
- **Branding**: Full transition to "Wolfclaw" identity with "Crafted by Pravin A Mathew" attribution.
- **Security**: Strengthened path sanitization and added explicit cybersecurity warnings.
- **API**: Refactored to FastAPI for increased performance and reliability.

### Fixed
- Session retrieval logic regressions.
- Workspace-to-User mapping inconsistencies.
- LLM engine initialization errors.

## [1.0.0] - 2025-12-15

Initial private release.

### Added
- Basic bot creation and chat.
- CLI-based agent execution.
- Integration with early OpenAI models.
