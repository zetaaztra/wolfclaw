# Packaging Instructions for Wolfclaw (Phase 5)

## Windows `.exe` via PyInstaller

1. **Install PyInstaller:**
   ```powershell
   pip install pyinstaller
   ```

2. **Build `wolfclaw` (FastAPI/Static Frontend):**
   ```powershell
   # From the wolfclaw/ directory
   pyinstaller --name "Wolfclaw Desktop" --noconfirm --onedir --windowed --add-data "static;static" --hidden-import "pynput" --hidden-import "pyautogui" --hidden-import "googleapiclient" --hidden-import "slack_sdk" --hidden-import "PIL" --hidden-import "openai" desktop_launcher.py
   ```

3. **Build `wolfclaw_legacy` (Streamlit Frontend):**
   ```powershell
   # From the wolfclaw_legacy/ directory
   pyinstaller --name "Wolfclaw Streamlit" --noconfirm --onedir --windowed --add-data "ui;ui" --add-data "core;core" --hidden-import "streamlit" desktop_launcher.py
   ```

*(Note: Add `--icon=static/favicon.ico` if you have an icon file).*

## Linux `.deb` Package

1. **Prerequisites:**
   Ensure you have `dpkg-deb` installed (standard on Ubuntu/Debian).

2. **Directory Structure:**
   Create a staging directory:
   ```bash
   mkdir -p wolfclaw-1.0.0/DEBIAN
   mkdir -p wolfclaw-1.0.0/usr/local/bin/wolfclaw
   mkdir -p wolfclaw-1.0.0/usr/share/applications/
   mkdir -p wolfclaw-1.0.0/usr/share/icons/hicolor/512x512/apps/
   ```

3. **Control File:**
   Create `wolfclaw-1.0.0/DEBIAN/control`:
   ```
   Package: wolfclaw
   Version: 1.0.0
   Section: utils
   Priority: optional
   Architecture: amd64
   Maintainer: Your Name <you@example.com>
   Description: Wolfclaw AI Command Center
    Multi-agent local orchestrator with Universal App Integrations,
    Watch & Learn Macros, and a Plugin Marketplace.
   ```

4. **Copy Application Files:**
   Copy the `dist/Wolfclaw Desktop/` folder (generated by running Pyinstaller on Linux) into `wolfclaw-1.0.0/usr/local/bin/wolfclaw/`.

5. **Create Desktop Entry:**
   Create `wolfclaw-1.0.0/usr/share/applications/wolfclaw.desktop`:
   ```ini
   [Desktop Entry]
   Version=1.0
   Name=Wolfclaw
   Comment=AI Command Center
   Exec=/usr/local/bin/wolfclaw/Wolfclaw\ Desktop
   Icon=wolfclaw
   Terminal=false
   Type=Application
   Categories=Utility;
   ```

6. **Build the `.deb`:**
   ```bash
   dpkg-deb --build wolfclaw-1.0.0
   ```
   This will output `wolfclaw-1.0.0.deb` which can be installed with `sudo apt install ./wolfclaw-1.0.0.deb`.
